# Running Interview Master

This project is set up using **Vite** and **React**. Follow these steps to run it locally:

## 1. Set Up Your API Key

The application requires a **Gemini API Key**.

1. Copy `.env.example` to `.env`.
2. Open `.env` and replace `your_gemini_api_key_here` with your actual API key from [AI Studio](https://aistudio.google.com/app/apikey).

## 2. Start the Development Server

// turbo
Run the following command in your terminal:

```bash
npm run dev
```

## 3. Access the Application

Once the server is running, open your browser to [http://localhost:5173/](http://localhost:5173/).
